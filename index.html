Esombley (Esoteric Assembly)


Commands (case insensitive):

INP x:Takes input at memory location x

OUT x:Takes output at memory location x

ADD x y:Sets memory loc. x to x+y & y to 0

SUB x y:Sets memory loc. x to x-y & y to 0

MULT w x y z:Sets w to 0 & z to w*x.

For MULT to work, y & z have to be 0 before multiplying

Commands seperated by semicolon & parameters by spaces
